void SetLeds ( int  state)

Sets the state of the BeagleBoard status LEDs.

Useful mostly for simple debugging.

Parameters:
stateThe state of the LEDs. The first bit defines the state for one LED, and the second bit the other. 0 makes both LEDs turn off, 1 and 2 turns on one or the other LED, and 3 turns on both.
