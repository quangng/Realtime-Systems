See the OMAP3 documentation for more information about how to use these.

Enable interrupts globally.

void InstallInterruptHandlers ( )

Install the IRQ multiplexer.

Allows separate handlers to be installed for all the numbered IRQs. Also sets all interrupt mask bits, disabling all interrupt sources, and acknowledges any pending interrupts. Must be called at the start of the program before trying to use interrupts.

void SetIRQHandler ( int  irq,
void(*)()  func 
)

Set the handler subroutine for a given IRQ.

This function will be called when this specific IRQ occurs.

Parameters:
irqThe number of the IRQ.
funcThe function that will handle this IRQ. This function will be called in an interrupt context with the (not very big) interrupt stack. It should finish quickly, and avoid using much stack space. It should also make sure to acknowledge the interrupt flag in whatever peripheral caused the IRQ, or else it will get called again immediately when it exits.
