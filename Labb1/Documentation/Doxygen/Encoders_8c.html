void InitEncoders ( )

Configure hardware for reading the encoders.

Calling this function also sets up the interrupt handler, and initializes all positions to 0. Note that InstallInterruptHandlers() also needs to be called for the interrupts to work.

Remarks:
Timing: Does not use external buses.

int ReadEncoderChange ( int  n)

Read the change in the encoder position since the previous call.

Parameters:
nThe number of the encoder to read, from 0 to 2.
Returns:
The difference in the position of the encoder since the last time this function was called for this encoder. Positive changes mean a clockwise turn, negative changes a counter-clockwise turn.
Remarks:
Timing: Does not use external buses.
