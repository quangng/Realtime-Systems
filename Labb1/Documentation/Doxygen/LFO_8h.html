int RunTriangleLFO ( LFO *  lfo)

Run an LFO to generate triangle-wave output.

A triangular LFO will produce output that increases linearly from 0 to the configured amplitude of the LFO, then decreases linearly to the negative value of the amplitude, and then increases linearly back to 0, which it will reach after the number of invocations specified for the period in InitLFO().

Parameters:
lfoA pointer to the LFO data structure to use.
