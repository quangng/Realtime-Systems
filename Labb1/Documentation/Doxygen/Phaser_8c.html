Initialize a phaser effect implemented using a delay.

Parameters:
phaserA pointer to the phaser effect data structure.
delayThe delay to apply to the audio signal, in samples.
mixThe mixing factor for the direct and delayed signals. The range is 0-0x10000, where 0x10000 is maximum effect and 0 is no effect.
mixThe feedback factor for the feeding the output back into the delayed signal. The range is 0-0x10000, where 0x10000 is full feedback and 0 is no feedback.

Run the phaser effect.

The input signal is mixed with a delayed version of earlier input, and written to the output buffer.

Parameters:
phaserA pointer to the phaser effect data structure.
inThe input sample buffer.
outThe output sample buffer.
numsamplesThe number of samples to process. The in and out buffers should contain this many samples.
